Featured Project | Stocker

A real-time stock tracking application using Django REST Framework for the API, Finnhub's WebSocket for live market updates, Celery/Redis for scheduled API requests, PostgreSQL for database, a React frontend with a market dashboard style, all deployed on a VPS using Docker and nginx.

Tech Stack

Django, Django REST Framework, Finnhub API WebSocket, PostgreSQL, Celery/Redis, React, TailwindCSS, Chart.js, Docker, nginx

Architecture Overview

The core of the project is the Django REST API which uses scheduled tasks to populate the database with intraday & EOD market data which the React App consumes to generate line graphs. The Django API is connected to Finnhub's websocket where it then streams the data into the React App using Django Channels for real-time market data display.
