Featured Project | Stocker

A real-time stock tracking application using the Django REST Framework for the API, Finnhub's WebSocket for live market updates, Celery/Redis for scheduled API requests, PostgreSQL for database, a React frontend with a market dashboard style, all deployed on a VPS using Docker and nginx.

Tech Stack

Django, Django REST Framework, Finnhub API WebSocket, PostgreSQL, Celery/Redis, React, TailwindCSS, Chart.js, Docker, nginx

Architecture Overview

The core of the project is the Django REST API which uses scheduled tasks to populate the database with intraday & EOD market data which the React consumes to generate line graphs. It also connects to Finnhub's websocket which streams real-time market data into my Django API which then streams it to the React frontend to display real-time market data.
